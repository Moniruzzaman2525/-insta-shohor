How does Javascript work:

JavaScript is a client-side scripting programming language, which means it has a single call stack. JavaScript can do one thing at a time. A call stack is a data structure that basically records where we are. If we set foot on a function, we place it on top of the stack. A popular example of a JavaScript engine is the V8 engine.JavaScript was invented and created by ECMA International, by Brendan Eich. The official name of JavaScript was ECMAScript.

The difference between local storage and session storage:

Single value can be easily stored in local storage and session storage . If you want to place object or array in local storage and session storage, you have to keep JSON.stringify () and after placing it it becomes string. And if you want to use it, you have to use JSON.parse () . Some differences between local storage and session storage : Local storage does not work for any specific tab, it works for any tab and Session storage works for a specific tab and when the tab is closed data is deleted . Local storage data has to be deleted manually but Closing the tab clears the session storage .
